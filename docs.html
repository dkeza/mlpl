Documentation

read: This command enables you to supply an integer for your program to process. The value you supply is placed in a variable.

read x

The instruction above read a integer user types on a console and places it into variable x. If this instruction is not the last instruction in the block it has to be followed by a semicolon. If the instruction is the last instruction in the block it must not be followed by a semicolon.

write: This command enables you to write out a variable, integer or a string to the console.

write x

The instruction above writes a value of the variable to the console.

write 5

The instruction above writes a value of the variable to the console.

write "message"

The instruction above writes a value of the variable to the console.

If this instruction is not the last instruction in the block it has to be followed by a semicolon. If the instruction is the last instruction in the block it must not be followed by a semicolon.
